# Security Policy

## Reporting a Vulnerability

Please **do not** file a public GitHub issue for security vulnerabilities.

Use GitHub's [private vulnerability reporting](https://github.com/ahed-isir/convo-recall/security/advisories/new) to submit a report. If that link is unavailable, email **adwansubs@icloud.com** with subject `[convo-recall security]` — do not include sensitive details in the initial email; wait for a secure channel to be established.

I aim to respond within a week.

Please include:
- A description of the vulnerability and its potential impact
- Steps to reproduce
- Any suggested mitigations (optional)
